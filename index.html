Who am I?

สวัสดีครับ ผมชื่อ ฐิตินันท์ รัตนพราว ตอนนี้อายุ 18 ปี เป็นนักเรียน ม.6/2 ที่ชอบในการเขียนโปรแกรมและพัฒนาเว็บไซต์ ชอบลองอะไรใหม่ ๆ ทั้งทางด้านเทคนิคและดีไซน์ และกำลังพัฒนาทักษะตัวเองเพื่อก้าวไปสู่สาย Developer เต็มตัวในอนาคต

ผมเริ่มต้นจากการลองเล่น Python , Lua และเขียนเว็บเล็ก ๆ ด้วย HTML / CSS ก่อนจะค่อย ๆ ขยับมาศึกษา JavaScript, C/C++, และเทคโนโลยีอื่น ๆ ที่เกี่ยวกับเว็บแอปพลิเคชัน ผมชอบความรู้สึกตอนที่ไอเดียบนกระดาษกลายเป็นโปรเจกต์ที่คนอื่นใช้งานได้จริง

นอกจากด้านเทคนิค ผมยังสนใจเรื่องการออกแบบให้ใช้งานง่าย ชอบทำ UI ที่ดูสะอาดตาแต่ยังมีลูกเล่นและแอนิเมชันที่ลื่นไหล ซึ่งเป็นเหตุผลที่ผมชอบเล่นกับ TailwindCSS, animation ต่าง ๆ และการจัด layout
